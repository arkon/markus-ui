This is the official UI style guide for the MarkUs Project.

You'll find various core elements used around MarkUs on this page, along with helpful code snippets so you can use them easily. Please adhere to the style shown on this page in order to keep MarkUs as cohesive as possible.

The production version of the styles for the core elements shown on this page can be downloaded here as a package of SCSS partial files and required JavaScript files. You can simply import _core.scss into another SCSS file! You'll also need to include the JavaScript files into the head of your HTML files.

If you're interested in seeing the code, you're welcome to check out the GitHub repo.

Last updated June 2014.

Regular text

Body: 16px baseline, 1.5em (24px) line height, 1.125em (18px) paragraph, normal weight (400).

Make sure to include the Open Sans web font! You can just add this into your HTML head: <link rel='stylesheet' href='http://fonts.googleapis.com/css?family=Open+Sans:400,300,600'>.

Links are shown in blue.

Code

<pre>Code should be shown in a monospace font like this. Use an additional plugin of your choice to perform syntax highlighting and HTML cleanup.</pre>

You can show inline code using <span class='code'> the code class </span>.
