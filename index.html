This is the official UI style guide for the MarkUs Project.

You'll find various core elements used around MarkUs on this page, along with helpful code snippets so you can use them easily. Please adhere to the style shown on this page in order to keep MarkUs as cohesive as possible. You're also free to use the styles provided here in any of your own projects.

The assets for the elements shown on this page can be downloaded here as a package of SCSS partials, images, and JavaScript files. You'll need to make sure that the JavaScript files are included in the head of your HTML files and the image references in _markus.scss are correct. You can then simply import _markus.scss into another SCSS file (@import 'markus';).

If you're interested in seeing the code, you're welcome to check out the GitHub repo. You can also find older releases here.

Last updated June 11 2014.

Regular text

Body: 16px baseline, 1.5em (24px) line height, 1.125em (18px) paragraph, normal weight (400).

Make sure to include the Open Sans web font! You can just add this into your HTML head: <link rel='stylesheet' href='http://fonts.googleapis.com/css?family=Open+Sans:400,300,600'>.

Links are shown in blue.

Tab pane

  • Tabs!
  • HTML Markup
  • JavaScript

This is based on jQuery UI's tabs widget, so be sure to include jquery-ui.js in your page.

Make sure the tab anchors' hash references correspond to the ids of the tab content divs! 

<div class='tab-pane' id='tabs'>
  <ul>
    <li><a href='#tab-1'><span>Tabs!</span></a></li>
    <li><a href='#tab-2'><span>HTML Markup</span></a></li>
    <li><a href='#tab-3'><span>JavaScript</span></a></li>
  </ul>
  <div id='tab-1'>
    <p>This is based on jQuery UI's tabs widget.</p>
  </div>
  <div id='tab-2'>
    <p>Some more content.</p>
  </div>
  <div id='tab-3'>
    <p>Lorem ipsum.</p>
  </div>
</div>

Simply instantiate the tabs like so: 

$('#tabs').tabs();

This is a dialog modal

The dialog has a box shadow of #8d8d8d at a size of 10px from the origin.

Make sure to include js/jquery.easyModal.js if you want to use this!

  </section>

Check out this awesome example dialog!

The close button should have a class called close. You can also click on the overlay behind this dialog modal to close it.

The modal windows should have a defined width. 

$('.dialog-selector').easyModal();
$('#open-link-selector').click(function() {
  $('.dialog-selector').trigger('openModal');
